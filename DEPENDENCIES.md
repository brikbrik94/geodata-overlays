# Abhängigkeiten: geodata-overlays

Dieses Plugin benötigt folgende Tools auf dem Host-System:

- **bash**: Orchestrierung
- **python3**: Generierung von Styles und Manifest
- **docker**: Zum Bauen der PMTiles (via Tippecanoe/Planetiler)
- **osmium**: Validierung von Daten (optional)
- **git**: Synchronisierung der GeoJSON-Quelldaten
